== Description ==

This plugin allows you to pay site contributors through the WordPress dashboard as you publish articles.

An OutVoice account is required. <a href="https://outvoice.com/sign-up">Sign up here</a>.

OutVoice let's you easily manage and pay content contributors. Watch this short video and get more information at <a href="https://outvoice.com">outvoice.com</a>.
